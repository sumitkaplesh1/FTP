# Contributing Guidelines

We welcome suggestions to improve our policy files.

## How to Contribute
1. Fork this repository.
2. Create a new branch for your changes.
3. Edit the policy file (use clear headings and Markdown formatting).
4. Commit your changes with a descriptive message.
5. Open a Pull Request to the `main` branch.

## Review Process
- All PRs will be reviewed before merging.
- Please keep changes focused and explain the reason in your PR description.
